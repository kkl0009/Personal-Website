Smart Pantry Mobile Application

Github repository Our system manual Our final presentation

About: This is a cross-platform mobile application which was designed to help elderly individuals keep track of their food items to allow them to be more independent. I was a lead programmer on our team of 5, and this project is the result of nearly a year of work (from August 2021 to May 2022). We made use of Amazon Web Services and Node.js to create the app. I highly recommend checking out our system manual up above. It includes all of our code, as well as screenshots of a near-final product. We ultimately received an A on this project, as well as all of its supplementary material.

Operating System Project

Github repository Our user manual

About: This is a command line-based operating system I programmed over the course of a semester with 3 other group members. The OS has several functionalities, such as the ability to interpret and carry-out user commands, the ability to schedule and load processes, the ability to perform its own memory management, the ability to use interrupt driven I/O, and some others. We completed this assignment for our Operating Systems class, and were one of the only groups to successfully implement the interrupts into our system.

Completed: Fall 2021

Skills used:

  • C
  • Command Handling
  • Pointers
  • Process Control Blocks
  • Memory Management
  • Interrupt Driven I/O
  • CLI
  • Assembly Language

Assembly Hangman Game

Github repository Our final report Our final presentation

About: This is a Hangman game I made on a team of 4 in MS-DOS using x86 Assembly Language. It uses simplified but stylish ASCII art, and has an efficient algorithm for checking letters in the game using AND/OR logic. While this program could have been implemented much more easily using another programming language, the challenge of using assembly ended up being very educational. I strongly encourage you to check out our report linked above, or you may instead prefer to watch a video of my group's final presentation. For reference, I am the second speaker in our presentation.

Date: Spring 2021

Skills used:

  • x86 Assembly Language
  • MS-DOS
  • Registers
  • Input Parsing
